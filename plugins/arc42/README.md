# arc42

Generate [arc42](https://arc42.org) architecture documentation from a codebase.

`/arc42:generate` scans the repo into a structured evidence base, then writes the
12 arc42 sections — Mermaid diagrams where the code gives high-confidence
structure, and explicit GAP flags where human input is needed (never fabricated).
